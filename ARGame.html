Augmented Reality Game - Whack-A-Mole

Demo

Role: Team Lead, Programmer    Timeline: 1 Month    Tools: Unity, C#, Maya

Overview

Whack-A-Jackson is a Whack-A-Mole inspired game developed using Augmented Reality. This application was created by a small group of four, over a month-long period using Unity. This was our teams' first experience with Augmented Reality before, so creating this application was a great learning experience for the team.

Contributions

  • As team lead I was in charge of holding weekly SCRUM meetings
  • Created paper prototypes of, and designed, the main menu
  • Developed various C# scripts for the main menu buttons, timer, and score counter
  • Set up the Unity file to perform Augmented Reality capabilities
  • Utilized and placed assets in order to create the main play area
